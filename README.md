# 🛡️ AI Fact-Check Agent

An AI-powered web application that extracts claims from PDF documents and verifies their accuracy using AI.

## 🚀 Features

* 📄 Upload PDF files
* 🔍 Extract text and factual claims
* 🤖 AI-based claim verification
* ✅ Detect Verified, Inaccurate, and False claims
* 📊 Display verification statistics
* 📥 Download results as CSV

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* PDF.js
* Claude API
